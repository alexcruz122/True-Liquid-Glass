This is just a Demo

Features

Liquid Glass Cards: Frosted glass effect with subtle depth and shadow for a modern UI look.

Interactive Movement: Cards respond to mouse movement for a parallax effect.

Background Video: Full-screen autoplay, muted, looped video for a dynamic look.

Responsive Design: Fully optimized for desktop, tablet, and mobile screens.

SVG Distortion Filter: Adds subtle frosted glass distortion for a realistic effect.

Technologies Used

HTML5

CSS3 (Flexbox, Media Queries, Backdrop Filter)

JavaScript (Mouse move interactions)

SVG Filters for glass distortion

Getting Started

Clone the repository:

git clone https://github.com/your-username/redtrex-landing-page.git


Open index.html in your preferred browser.

Replace the background video (StyleLogo/overwatch-2-x-le-sserafim.1920x1080.mp4) with your own video if needed.

Update the text content in .liquid-glass cards to match your own company info.

Customization

Cards: You can add or remove .liquid-glass divs to include more sections.

Colors & Glass Effect: Adjust the backdrop-filter, box-shadow, and rgba() values in CSS for different transparency or blur effects.
